# 🌊 Swell-Time Surf School Website

A simple and responsive surfing school website built using **HTML, CSS, and JavaScript** for learning and practice purposes.

---

## 📌 Features

- Responsive navigation bar
- Hero section with background image
- Surf lesson cards
- About and Contact sections
- Mobile menu toggle using JavaScript
- Clean and beginner-friendly code structure

- CSS3
- JavaScript
